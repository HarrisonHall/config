# config
Run `setup.sh` in order to set up the system.  
Some things need to be set up manually. 

### Scripts 
These are 
* audio
* audiovisualizer
* brightness
* colorgrab
* files
* lockscreen
* music
* network
* screenshot
* sleepy
* suspend
* volume
  * Set volume in 20% intervals
  * Alternatively one can input 0-1
